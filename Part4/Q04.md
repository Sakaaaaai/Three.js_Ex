# Part4 | マウス位置に反応する波紋シェーダーを実装しよう

## 目標

ShaderMaterialを使い、頂点シェーダーで頂点の高さを操作し、マウス位置を起点に時間とともに減衰する波紋アニメーションを実装できるようになる。

## 今回の演習で学ぶこと

### ShaderMaterialとuniforms

既製のMaterial(MeshStandardMaterialなど)では表現できない見た目を作りたいとき、`THREE.ShaderMaterial`を使い、自分でGLSL(頂点シェーダー・フラグメントシェーダー)を書きます。JavaScript/TypeScript側からGLSLへ値を渡すには`uniforms`を使います。今回は次の3つのuniformsを使います。

- `uTime`: 経過時間(秒)。時間とともに変化するアニメーションに使う
- `uMouse`: マウスが指している平面上のUV座標(0〜1)
- `uAmplitude` / `uFrequency`(チャレンジ課題): 波紋の振幅・周波数をGUIから調整するための値

### Vertex ShaderとFragment Shaderの役割

- **Vertex Shader**: 頂点ごとに実行され、頂点の位置(`gl_Position`)を計算する。今回は「マウスからの距離に応じて頂点のZ座標を持ち上げる」ことで波紋の凹凸を作る
- **Fragment Shader**: ピクセルごとに実行され、色(`gl_FragColor`)を計算する。今回は波紋の高さ(`vElevation`)に応じて色を変化させる
- 2つのシェーダー間で値を受け渡すのが`varying`です。頂点シェーダーで計算した`vElevation`を、フラグメントシェーダー側で色の計算に使います

### マウス座標を平面上のUV座標に変換する

スクリーン座標(ピクセル)をそのままGLSLに渡しても、平面のどこを指しているかは分かりません。そこで`THREE.Raycaster`を使い、「マウスの位置から見て、平面のどこと交差するか」を計算します。交差判定の結果(`intersection.uv`)には、その交差点の平面上のUV座標(0〜1)が自動的に入っているため、これをそのまま`uMouse`に渡すことができます。

### 波紋の計算式の考え方

波紋は「距離が近いほど・時間が経つほど、振動しながら減衰する」現象です。GLSLでは次のように表現できます。

```glsl
float dist = distance(uv, uMouse);              // マウスからの距離
float wave = sin(dist * 周波数 - uTime * 速さ);   // 振動(距離と時間で位相がずれる)
float decay = exp(-dist * 減衰係数);              // 距離が離れるほど小さくなる減衰
float elevation = wave * decay * 振幅;
```

## 問題文

`src/main.ts`には、マウス位置の追跡・`uTime`の更新・シェーダーの読み込みなど、JS/TS側の配線がすべて実装済みです。未実装なのは、以下の2つのシェーダーファイルの中身(GLSL)です。

## ヒント

- `distance(uv, uMouse)`はGLSL組み込み関数で、2つのvec2間の距離を計算します
- `sin`関数の位相に`uTime`を掛けた項を入れると、時間とともに波紋がアニメーションします
- `exp(-dist * 係数)`は距離が離れるほど急激に0へ近づく減衰カーブです。係数を大きくするほど波紋の広がりが小さくなります
- フラグメントシェーダーでは`mix(colorA, colorB, t)`を使うと、`t`(0〜1を想定)に応じて2色を補間できます。`vElevation`は正負に変動する小さな値なので、`vElevation * 3.0 + 0.5`のようにスケール・オフセットしてから`mix`のtとして使うと調整しやすいです

## よくあるエラー

- 頂点シェーダーで`pos.z`に加算する前の`position`をそのまま`gl_Position`の計算に使ってしまい、波紋が反映されない → 加算後の`pos`を使う
- `vElevation`に値を代入する前に`gl_Position`を計算してしまう(GLSLは上から順に実行されるため、代入順序に注意)
- 平面の分割数(`PlaneGeometry`の第3・第4引数)が少なすぎて、波紋がカクカクした見た目になる → 分割数を増やす(ただし増やしすぎると頂点数が増え負荷が上がる)
- `uMouse`の初期値を`(0, 0)`のままにしていて、初回描画時に画面端に不自然な波紋が出る → 初期値は平面の中心付近(`0.5, 0.5`)にしておく

```ts
export const rippleVertexShader = /* glsl */ `
uniform float uTime;
uniform vec2 uMouse;
uniform float uAmplitude;
uniform float uFrequency;

varying vec2 vUv;
varying float vElevation;

void main() {
  vUv = uv;

  vec3 pos = position;

  float dist = distance(uv, uMouse);
  float elevation = sin(dist * uFrequency - uTime * 4.0) * exp(-dist * 6.0) * uAmplitude;
  pos.z += elevation;
  vElevation = elevation;

  gl_Position = projectionMatrix * modelViewMatrix * vec4(pos, 1.0);
}
`;
```

```ts
export const rippleFragmentShader = /* glsl */ `
varying vec2 vUv;
varying float vElevation;

void main() {
  vec3 deepColor = vec3(0.1, 0.2, 0.5);
  vec3 crestColor = vec3(0.6, 0.9, 1.0);

  vec3 color = mix(deepColor, crestColor, vElevation * 3.0 + 0.5);

  gl_FragColor = vec4(color, 1.0);
}
`;
```

<summary>模範解答のコードを見る(src/answer/main.ts、チャレンジ課題のGUI実装込み)</summary>

```ts
import * as THREE from 'three';
import GUI from 'lil-gui';
import '../style.css';
import { rippleVertexShader } from './shaders/rippleVertex';
import { rippleFragmentShader } from './shaders/rippleFragment';

const scene = new THREE.Scene();

const camera = new THREE.PerspectiveCamera(
  50,
  window.innerWidth / window.innerHeight,
  0.1,
  100
);
camera.position.set(0, 2.4, 3.2);
camera.lookAt(0, 0, 0);

const renderer = new THREE.WebGLRenderer({ antialias: true });
renderer.setSize(window.innerWidth, window.innerHeight);
renderer.setPixelRatio(window.devicePixelRatio);
document.getElementById('app')?.appendChild(renderer.domElement);

const geometry = new THREE.PlaneGeometry(4, 4, 200, 200);
geometry.rotateX(-Math.PI / 2);

const uniforms = {
  uTime: { value: 0 },
  uMouse: { value: new THREE.Vector2(0.5, 0.5) },
  uAmplitude: { value: 0.15 },
  uFrequency: { value: 40 },
};

const material = new THREE.ShaderMaterial({
  uniforms,
  vertexShader: rippleVertexShader,
  fragmentShader: rippleFragmentShader,
  side: THREE.DoubleSide,
});

const plane = new THREE.Mesh(geometry, material);
scene.add(plane);

const raycaster = new THREE.Raycaster();
const pointerNdc = new THREE.Vector2();

window.addEventListener('pointermove', (event) => {
  pointerNdc.x = (event.clientX / window.innerWidth) * 2 - 1;
  pointerNdc.y = -(event.clientY / window.innerHeight) * 2 + 1;

  raycaster.setFromCamera(pointerNdc, camera);
  const intersections = raycaster.intersectObject(plane);

  if (intersections.length > 0 && intersections[0].uv) {
    uniforms.uMouse.value.copy(intersections[0].uv);
  }
});

window.addEventListener('resize', () => {
  camera.aspect = window.innerWidth / window.innerHeight;
  camera.updateProjectionMatrix();
  renderer.setSize(window.innerWidth, window.innerHeight);
});

const gui = new GUI();
gui.add(uniforms.uAmplitude, 'value', 0, 0.4, 0.01).name('amplitude');
gui.add(uniforms.uFrequency, 'value', 5, 80, 1).name('frequency');

const clock = new THREE.Clock();

function animate(): void {
  requestAnimationFrame(animate);
  uniforms.uTime.value = clock.getElapsedTime();
  renderer.render(scene, camera);
}

animate();
```
