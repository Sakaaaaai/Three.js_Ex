# Part3 | 疑似深度センサーデータを読み込む「深度カメラビューア」を作ろう

## 目標

CSVで配布された点群データをThree.jsに読み込み、深度(intensity)値に応じた色分け表示と、マウス操作(回転・ズーム)ができる3Dビューアを構築できるようになる。

## 今回の演習で学ぶこと

### Points / PointsMaterialの基礎

`THREE.Mesh` が「面」を描画するのに対し、`THREE.Points` は「点」を1つずつ描画するオブジェクトです。数千〜数十万点の点群データをMeshとしてポリゴン化するのは非常にコストが高いため、点群の可視化には`Points`が適しています。

`PointsMaterial`には、点のサイズを決める`size`、カメラ距離に応じて点を小さく見せる`sizeAttenuation`、頂点ごとに異なる色を使う`vertexColors`などのプロパティがあります。

### CSVをPapaParseで読み込む

配布される疑似深度データ(`src/data/depth-scan.csv`)は、`x,y,z,intensity`のヘッダーを持つCSVです。PapaParseの`header: true`オプションを使うと、1行1オブジェクトの配列としてパースできます。`dynamicTyping: true`を指定すると、文字列ではなく数値としてパースされます。

このCSVは実際の研究データではなく、`scripts/generate-sample-data.mjs`というスクリプトで生成した合成データです(`npm run generate:data`で再生成できます)。

### 深度値・強度値によるカラーマッピング

センサーの生データ(数値)をそのまま眺めても直感的に理解しづらいため、値を色に変換して可視化します。今回のサンプルデータでは`intensity`(0〜1の疑似的な反射強度)を使い、低い値を青、高い値を赤にマッピングする「ヒートマップ表現」を実装します。`THREE.Color.lerpColors()`を使うと、2色間を割合で補間した色を作れます。

### BufferGeometryへの位置・色属性の設定

`Points`に使う`BufferGeometry`には、`position`属性(x, y, z)と`color`属性(r, g, b)をそれぞれ`Float32BufferAttribute`として設定します。属性を設定した後は`computeBoundingSphere()`を呼び出してください。これを忘れると、OrbitControlsの操作性やカメラのカリング判定に影響が出ることがあります。

## ヒント

- `Papa.parse<Partial<DepthPoint>>(csvText, { header: true, dynamicTyping: true, skipEmptyLines: true })` の戻り値の`.data`が配列です
- パース結果の各行が本当に数値を持っているかは、型ガード関数(`row is DepthPoint`)を書いて`Array.prototype.filter`で絞り込むと、`any`を使わずに済みます
- `intensityToColor`は、`THREE.MathUtils.clamp`で値を0〜1に丸めてから、`new THREE.Color().lerpColors(colorA, colorB, t)`で補間すると実装しやすいです
- `OrbitControls`は`three/examples/jsm/controls/OrbitControls.js`からインポートします。`controls.enableDamping = true`にすると滑らかな操作感になりますが、その場合は毎フレーム`controls.update()`を呼ぶ必要があります
- `PointsMaterial`の`size`は、今回のデータのスケール感だと`0.05`前後がちょうど良いサイズです

## よくあるエラー

- `dynamicTyping: true`を指定し忘れ、`x`や`intensity`が文字列のままになり、位置がおかしくなる・色が全部同じになる
- `geometry.computeBoundingSphere()`を呼び忘れ、OrbitControlsでズームアウトしすぎると点群が見えなくなる、または操作がぎこちなくなる
- `controls.enableDamping = true`にしたのに、アニメーションループで`controls.update()`を呼んでおらず、慣性が効かない(操作直後にしか反映されない)
- `color`属性を`setAttribute`する際に`itemSize`を`3`にし忘れ、色がおかしくなる

```ts
import Papa from 'papaparse';

export interface DepthPoint {
  x: number;
  y: number;
  z: number;
  intensity: number;
}

function isDepthPoint(row: Partial<DepthPoint>): row is DepthPoint {
  return (
    typeof row.x === 'number' &&
    typeof row.y === 'number' &&
    typeof row.z === 'number' &&
    typeof row.intensity === 'number'
  );
}

export function parseDepthPoints(csvText: string): DepthPoint[] {
  const result = Papa.parse<Partial<DepthPoint>>(csvText, {
    header: true,
    dynamicTyping: true,
    skipEmptyLines: true,
  });

  return result.data.filter(isDepthPoint);
}
```

```ts
import * as THREE from 'three';

const LOW_COLOR = new THREE.Color(0x2b6cb0);
const MID_COLOR = new THREE.Color(0x38a169);
const HIGH_COLOR = new THREE.Color(0xe53e3e);

export function intensityToColor(intensity: number): THREE.Color {
  const clamped = THREE.MathUtils.clamp(intensity, 0, 1);
  const color = new THREE.Color();

  if (clamped < 0.5) {
    color.lerpColors(LOW_COLOR, MID_COLOR, clamped / 0.5);
  } else {
    color.lerpColors(MID_COLOR, HIGH_COLOR, (clamped - 0.5) / 0.5);
  }

  return color;
}
```

```ts
import * as THREE from 'three';
import type { DepthPoint } from '../utils/parseDepthCsv';
import { intensityToColor } from '../utils/colormap';

export class DepthPointCloud {
  readonly points: THREE.Points<THREE.BufferGeometry, THREE.PointsMaterial>;

  constructor(depthPoints: DepthPoint[]) {
    const geometry = new THREE.BufferGeometry();

    const positions = new Float32Array(depthPoints.length * 3);
    const colors = new Float32Array(depthPoints.length * 3);

    depthPoints.forEach((point, i) => {
      positions[i * 3] = point.x;
      positions[i * 3 + 1] = point.y;
      positions[i * 3 + 2] = point.z;

      const color = intensityToColor(point.intensity);
      colors[i * 3] = color.r;
      colors[i * 3 + 1] = color.g;
      colors[i * 3 + 2] = color.b;
    });

    geometry.setAttribute('position', new THREE.Float32BufferAttribute(positions, 3));
    geometry.setAttribute('color', new THREE.Float32BufferAttribute(colors, 3));
    geometry.computeBoundingSphere();

    const material = new THREE.PointsMaterial({
      size: 0.05,
      vertexColors: true,
      sizeAttenuation: true,
    });

    this.points = new THREE.Points(geometry, material);
  }
}
```

```ts
import * as THREE from 'three';
import { OrbitControls } from 'three/examples/jsm/controls/OrbitControls.js';

export interface SceneContext {
  scene: THREE.Scene;
  camera: THREE.PerspectiveCamera;
  renderer: THREE.WebGLRenderer;
  controls: OrbitControls;
}

export function createScene(container: HTMLElement): SceneContext {
  const scene = new THREE.Scene();
  scene.background = new THREE.Color(0x101010);

  const camera = new THREE.PerspectiveCamera(
    60,
    window.innerWidth / window.innerHeight,
    0.1,
    100
  );
  camera.position.set(0, 3, 8);

  const renderer = new THREE.WebGLRenderer({ antialias: true });
  renderer.setSize(window.innerWidth, window.innerHeight);
  renderer.setPixelRatio(window.devicePixelRatio);
  container.appendChild(renderer.domElement);

  const controls = new OrbitControls(camera, renderer.domElement);
  controls.enableDamping = true;

  window.addEventListener('resize', () => {
    camera.aspect = window.innerWidth / window.innerHeight;
    camera.updateProjectionMatrix();
    renderer.setSize(window.innerWidth, window.innerHeight);
  });

  return { scene, camera, renderer, controls };
}
```
