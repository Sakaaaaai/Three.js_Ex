# Part5 | 点群ビューアにBloom(発光)効果を追加しよう

## 目標

EffectComposerを使ったポストプロセッシングの基本構成を理解し、Part3で作った点群ビューアにBloom効果を追加して、データの見せ方を強化できるようになる。

## 今回の演習で学ぶこと

### ポストプロセッシングとは何か

通常、Three.jsは`renderer.render(scene, camera)`でシーンを直接画面に描画します。ポストプロセッシングは、一度レンダリングした画像に対して、さらに画像処理的なエフェクト(発光・被写界深度・環境遮蔽など)を重ねる仕組みです。`EffectComposer`が、これらのエフェクト(Pass)をチェーンのようにつなげる役割を持ちます。

- `RenderPass`: 通常のシーン描画を行うPass(必ず最初に追加する)
- `UnrealBloomPass`: 明るい部分をにじませて発光させるPass

ポストプロセッシングを使うと、通常の描画に加えて追加のフルスクリーン描画が発生するため、描画コストが上がる点も覚えておいてください。

### Bloomの仕組みとパラメータ

Bloomは「輝度が高い部分を抽出し、ぼかして、元の画像に加算合成する」ことで発光しているように見せるエフェクトです。`UnrealBloomPass`には主に3つのパラメータがあります。

- `strength`: 発光の強さ
- `radius`: 発光がにじむ範囲
- `threshold`: この輝度を超えた部分だけが発光対象になる(しきい値)

`threshold`を上げるほど、明るい部分だけが選択的に発光するようになります。データ可視化においては、「意味のある部分だけを発光させる」ことが重要で、すべてを発光させると情報が読み取りにくくなります。

## ヒント

- `UnrealBloomPass`のコンストラクタは`(resolution: THREE.Vector2, strength, radius, threshold)`の順です
- `composer.setSize(width, height)`と`bloomPass.setSize(width, height)`(または`resolution`の更新)の両方をリサイズ時に呼ぶ必要があります
- `threshold`を`0.3`前後にすると、今回のカラーマップ(暖色系が明るい)ではちょうど暖色部分だけが発光しやすくなります。ぴったりの値は試行錯誤で見つけてください
- lil-guiで`strength`・`radius`・`threshold`をリアルタイムに調整できるようにすると、パラメータ探しが格段に楽になります

- `RenderPass`を追加し忘れ、画面が真っ黒になる(`UnrealBloomPass`だけでは何も描画されない)
- リサイズ時に`renderer.setSize`だけ更新し、`composer`側のサイズを更新し忘れて表示が歪む
- `threshold`を低くしすぎて、点群全体が白く発光してしまい、データが読み取れなくなる
- `animate()`内で`renderer.render(scene, camera)`を消し忘れ、`composer.render()`と二重に描画してしまう

```ts
import * as THREE from 'three';
import { OrbitControls } from 'three/examples/jsm/controls/OrbitControls.js';

export interface SceneContext {
  scene: THREE.Scene;
  camera: THREE.PerspectiveCamera;
  renderer: THREE.WebGLRenderer;
  controls: OrbitControls;
}

export function createScene(container: HTMLElement): SceneContext {
  const scene = new THREE.Scene();
  scene.background = new THREE.Color(0x050505);

  const camera = new THREE.PerspectiveCamera(
    60,
    window.innerWidth / window.innerHeight,
    0.1,
    100
  );
  camera.position.set(0, 3, 8);

  const renderer = new THREE.WebGLRenderer({ antialias: true });
  renderer.setSize(window.innerWidth, window.innerHeight);
  renderer.setPixelRatio(window.devicePixelRatio);
  container.appendChild(renderer.domElement);

  const controls = new OrbitControls(camera, renderer.domElement);
  controls.enableDamping = true;

  return { scene, camera, renderer, controls };
}
```

```ts
import * as THREE from 'three';
import { EffectComposer } from 'three/examples/jsm/postprocessing/EffectComposer.js';
import { RenderPass } from 'three/examples/jsm/postprocessing/RenderPass.js';
import { UnrealBloomPass } from 'three/examples/jsm/postprocessing/UnrealBloomPass.js';

export interface BloomComposer {
  composer: EffectComposer;
  bloomPass: UnrealBloomPass;
  setSize: (width: number, height: number) => void;
}

export function createBloomComposer(
  renderer: THREE.WebGLRenderer,
  scene: THREE.Scene,
  camera: THREE.PerspectiveCamera
): BloomComposer {
  const composer = new EffectComposer(renderer);
  composer.addPass(new RenderPass(scene, camera));

  const bloomPass = new UnrealBloomPass(
    new THREE.Vector2(window.innerWidth, window.innerHeight),
    1.2,
    0.4,
    0.3
  );
  composer.addPass(bloomPass);

  const setSize = (width: number, height: number): void => {
    composer.setSize(width, height);
    bloomPass.setSize(width, height);
  };

  return { composer, bloomPass, setSize };
}
```

```ts
import GUI from 'lil-gui';
import '../style.css';
import rawCsv from '../data/depth-scan.csv?raw';
import { createScene } from './scene/createScene';
import { parseDepthPoints } from './utils/parseDepthCsv';
import { DepthPointCloud } from './entities/DepthPointCloud';
import { createBloomComposer } from './postprocessing/createBloomComposer';

const appElement = document.getElementById('app');
if (!appElement) {
  throw new Error('#app element not found');
}

const { scene, camera, renderer, controls } = createScene(appElement);

const depthPoints = parseDepthPoints(rawCsv);
const pointCloud = new DepthPointCloud(depthPoints);
scene.add(pointCloud.points);

const { composer, bloomPass, setSize } = createBloomComposer(renderer, scene, camera);

const gui = new GUI();
gui.add(bloomPass, 'strength', 0, 3, 0.05);
gui.add(bloomPass, 'radius', 0, 1, 0.01);
gui.add(bloomPass, 'threshold', 0, 1, 0.01);

window.addEventListener('resize', () => {
  camera.aspect = window.innerWidth / window.innerHeight;
  camera.updateProjectionMatrix();
  renderer.setSize(window.innerWidth, window.innerHeight);
  setSize(window.innerWidth, window.innerHeight);
});

function animate(): void {
  requestAnimationFrame(animate);
  controls.update();
  composer.render();
}

animate();
```
