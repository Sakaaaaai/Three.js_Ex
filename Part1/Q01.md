# Part1 | 「ボックスが回るだけ」コードを実務レベルの構成にリファクタリングしよう

## 目標

1ファイルにすべて書かれた「なんとなく動くThree.jsコード」を、役割ごとに分割された「保守できるプロジェクト構成」にリファクタリングできるようになる。

## 今回の演習で学ぶこと

### なぜファイルを分割するのか

`src/main.ts` に「シーン構築」「オブジェクト生成」「GUI操作」「リサイズ処理」「アニメーションループ」を全部書いても、Three.jsは問題なく動きます。しかし、これはオブジェクトの数や機能が増えるほど、次のような問題を引き起こします。

- どこに何が書いてあるか探すのが大変になる(可読性の低下)
- 1つの変更が別の場所に影響を与えないか不安になる(変更の安全性の低下)
- 同じようなオブジェクトを増やしたいときに、コピペを繰り返すことになる(再利用性の低下)

実務のThree.jsプロジェクトでは、責務(役割)ごとにファイルを分割するのが一般的です。今回は以下の4分割を採用します。

```
src/
  scene/     ← シーン全体で1つしか存在しないもの(Camera, Renderer, Lightなど)
  entities/  ← 個々の3Dオブジェクトをクラスとしてカプセル化したもの
  state/     ← UIとシーンで共有する状態(Pub/Subストア)
  utils/     ← リサイズ処理など、特定のオブジェクトに属さない共通処理
```

### Object3Dの型階層を意識する

Three.jsの `Mesh` は `THREE.Mesh<Geometry, Material>` のようにジェネリクスを持てます。`any` を使わず、`THREE.Mesh<THREE.BoxGeometry, THREE.MeshStandardMaterial>` のように型を明示すると、`mesh.material.color` のような操作でエディタの補完が効き、実行時エラーを未然に防げます。

### 状態管理(Pub/Sub)という考え方

GUIで変更した値を、直接 `mesh.material.color.set(...)` のように書き換えることもできますが、これだと「GUIがThree.jsのオブジェクトを直接知っている」状態になり、後からGUI以外の方法(WebSocket経由の値など)で色を変えたくなったときに困ります。

そこで、GUIは「状態(ストア)」だけを書き換え、3Dオブジェクト側は「状態の変更を購読(subscribe)して自分自身を更新する」という設計にします。この一方向の流れ(GUI → ストア → 3Dオブジェクト)が、後のPart(WebSocketでのリアルタイム更新など)でもそのまま使える設計の土台になります。

## 問題文

配布された `src/main.ts` は「ボックスが回り、GUIで色とサイズを変更できる」という機能がすべて実装済みですが、1ファイルにベタ書きされています。これを以下の構成にリファクタリングしてください。

1. `src/scene/createScene.ts` を作成し、Scene・Camera・Light・Rendererの構築処理を関数として切り出す
2. `src/entities/RotatingBox.ts` を作成し、ボックスの生成・回転・色とサイズの反映処理をクラスとして切り出す(`any` を使わないこと)
3. `src/state/store.ts` を作成し、`color` と `size` を保持するシンプルなPub/Subストアを実装する
4. `src/utils/resize.ts` を作成し、ウィンドウリサイズ処理を関数として切り出す
5. `src/main.ts` は上記4つを組み合わせて呼び出すだけの薄い初期化処理にする

### 完成イメージ

```
- ブラウザ画面いっぱいに、回転する箱(Box)が表示される
- 画面右上のGUIパネルで色を変更すると、箱の色がリアルタイムに変わる
- GUIパネルでサイズ(0.2〜3)を変更すると、箱の大きさがリアルタイムに変わる
- 見た目の動作自体はリファクタリング前後で一切変わらない
```

### 動作確認

1. 画面に回転するボックスが表示されていること
2. GUIパネルの色変更で、ボックスの色が変わること
3. GUIパネルのサイズ変更で、ボックスの大きさが変わること
4. ブラウザの幅を変えても、表示が崩れない(リサイズに追従する)こと
5. `tsc --noEmit` を実行して型エラーが出ないこと

## ヒント

- `createScene` は `scene` / `camera` / `renderer` をまとめたオブジェクトを返すようにすると、`main.ts` 側で受け取りやすくなります
- `RotatingBox` クラスは、コンストラクタで `boxStore` の初期値を読み込み、`boxStore.subscribe(...)` で以後の変更を受け取るようにします
- ストアの `setState` は、渡された値だけを部分的に更新できるようにする(`Partial<BoxState>` を受け取る)と、色だけ・サイズだけの変更にも対応できます
- GUIのコールバック(`onChange`)の中では、Three.jsのオブジェクトを直接触らず、`boxStore.setState(...)` を呼ぶだけにします

## 達成基準

- [ ] `main.ts` が「各モジュールを組み合わせて呼び出す」程度まで薄くなっている
- [ ] Scene/Camera/Renderer構築処理が `scene/createScene.ts` に分離されている
- [ ] ボックスの生成・更新ロジックが `entities/RotatingBox.ts` のクラスにカプセル化されている
- [ ] `color`・`size` の状態が `state/store.ts` のPub/Subストアで一元管理されている
- [ ] `any` を使わず、Meshやストアの型が明示されている
- [ ] リサイズ処理が `utils/resize.ts` に分離されている

## よくあるエラー

- `container.appendChild` で `appElement` が `null` の可能性を型エラーとして指摘される → `if (!appElement) throw new Error(...)` のようにnullチェックを入れる
- `RotatingBox` のコンストラクタでストアを購読する前に初期値を反映し忘れ、初期表示のサイズ・色がずれる → コンストラクタ内で `boxStore.getState()` を使って初期値を反映してから `subscribe` する
- GUIのコールバック内で直接 `mesh.material.color.set(...)` を書いてしまい、ストア経由になっていない → ストアの `setState` を呼ぶだけにする

```ts
import * as THREE from 'three';

export interface SceneContext {
  scene: THREE.Scene;
  camera: THREE.PerspectiveCamera;
  renderer: THREE.WebGLRenderer;
}

export function createScene(container: HTMLElement): SceneContext {
  const scene = new THREE.Scene();

  const camera = new THREE.PerspectiveCamera(
    75,
    window.innerWidth / window.innerHeight,
    0.1,
    100
  );
  camera.position.z = 5;

  const directionalLight = new THREE.DirectionalLight(0xffffff, 1);
  directionalLight.position.set(3, 3, 3);
  scene.add(directionalLight);
  scene.add(new THREE.AmbientLight(0xffffff, 0.3));

  const renderer = new THREE.WebGLRenderer({ antialias: true });
  renderer.setSize(window.innerWidth, window.innerHeight);
  renderer.setPixelRatio(window.devicePixelRatio);
  container.appendChild(renderer.domElement);

  return { scene, camera, renderer };
}
```

```ts
export interface BoxState {
  color: string;
  size: number;
}

type Listener = (state: BoxState) => void;

class BoxStore {
  private state: BoxState = { color: '#4f8fdb', size: 1 };
  private listeners = new Set<Listener>();

  getState(): BoxState {
    return this.state;
  }

  setState(partial: Partial<BoxState>): void {
    this.state = { ...this.state, ...partial };
    this.listeners.forEach((listener) => listener(this.state));
  }

  subscribe(listener: Listener): () => void {
    this.listeners.add(listener);
    return () => this.listeners.delete(listener);
  }
}

export const boxStore = new BoxStore();
```

```ts
import * as THREE from 'three';
import { boxStore } from '../state/store';

export class RotatingBox {
  readonly mesh: THREE.Mesh<THREE.BoxGeometry, THREE.MeshStandardMaterial>;

  constructor() {
    const initialState = boxStore.getState();

    const geometry = new THREE.BoxGeometry(1, 1, 1);
    const material = new THREE.MeshStandardMaterial({ color: initialState.color });

    this.mesh = new THREE.Mesh(geometry, material);
    this.mesh.scale.setScalar(initialState.size);

    boxStore.subscribe((state) => {
      this.mesh.material.color.set(state.color);
      this.mesh.scale.setScalar(state.size);
    });
  }

  tick(deltaRotation = 0.01): void {
    this.mesh.rotation.x += deltaRotation;
    this.mesh.rotation.y += deltaRotation;
  }
}
```

```ts
import * as THREE from 'three';

export function bindResize(camera: THREE.PerspectiveCamera, renderer: THREE.WebGLRenderer): void {
  window.addEventListener('resize', () => {
    camera.aspect = window.innerWidth / window.innerHeight;
    camera.updateProjectionMatrix();
    renderer.setSize(window.innerWidth, window.innerHeight);
  });
}
```

```ts
const appElement = document.getElementById('app');
if (!appElement) {
  throw new Error('#app element not found');

const { scene, camera, renderer } = createScene(appElement);

const box = new RotatingBox();
scene.add(box.mesh);

bindResize(camera, renderer);

const guiState = { ...boxStore.getState() };
const gui = new GUI();
gui.addColor(guiState, 'color').onChange((value: string) => {
  boxStore.setState({ color: value });
});
gui.add(guiState, 'size', 0.2, 3).onChange((value: number) => {
  boxStore.setState({ size: value });
});

function animate(): void {
  requestAnimationFrame(animate);
  box.tick();
  renderer.render(scene, camera);
}
```
