# Part8 | 総仕上げプロジェクト「センサーデータ3Dダッシュボード」

## 目標

Part1〜7で学んだ要素(プロジェクト構成・点群読み込み・カスタムシェーダー・ポストプロセッシング・WebSocketによるリアルタイム連携・GUI)を1つのアプリケーションに統合し、ポートフォリオとして提示できる成果物を完成させる。

## 今回の演習で学ぶこと

### 統合プロジェクトで最も重要なのは「組み合わせる順序」

総仕上げプロジェクトで難しいのは、個々の技術要素そのものよりも「どの順序で積み上げるか」です。今回のおすすめの実装順序は以下の通りです。

1. まずデータ(点群)が正しく表示される状態を作る(Part3の応用)
2. 見た目の表現(シェーダーによるヒートマップ)を作り込む(Part4の応用)
3. 見栄えを整える(Bloom、Part5の応用)
4. 最後にリアルタイム性(WebSocket、Part6の応用)を足す

見た目が仕上がっていない状態でリアルタイム更新を先に実装すると、「動いているはずなのに何が変わったか分からない」状態になりデバッグが難しくなります。土台を固めてから動きを足す、という順序が重要です。

### CPUで計算した色 vs シェーダーで計算した色

Part3では、点群の色をJavaScript側で計算し、`BufferAttribute`として都度書き込んでいました。今回はPart4の考え方を応用し、色の計算(ヒートマップ変換)をフラグメントシェーダー側に移しています。これにより、「センサー値が変わるたびにCPU側で全点の色を再計算する」という重い処理を避け、シェーダーのuniform(`uSensorBoost`)を1つ更新するだけで、GPU側が全ての点の色をリアルタイムに計算し直してくれます。

### 状態管理層を経由した統合設計

WebSocketで受信したセンサー値は、直接シェーダーのuniformを書き換えるのではなく、Part1・Part6と同じように「ストアを更新 → 購読しているコード側がuniformに反映する」という流れを踏襲しています。これにより、GUIの`updateMode`を`paused`にするだけで、リアルタイム更新の適用/停止をシンプルに切り替えられます。

## 問題文

`src/main.ts`には、CSV読み込み・WebSocket接続・状態管理(ストア)といった前提知識の部分は実装済みです。以下の5箇所のTODOを実装し、「センサーデータ3Dダッシュボード」を完成させてください。

1. `depthPoints`から`position`属性・`aIntensity`属性を持つ`BufferGeometry`を構築し、`heatmapVertexShader`/`heatmapFragmentShader`を使った`ShaderMaterial`で`THREE.Points`を作成してシーンに追加する(`uniforms`に`uSensorBoost`を含める)
2. `EffectComposer` + `RenderPass` + `UnrealBloomPass`をセットアップする
3. `sensorStore`を購読し、`updateMode`が`realtime`のときだけ`uSensorBoost`を更新する
4. リサイズ時に`composer`・`bloomPass`のサイズも更新する
5. アニメーションループで`composer.render()`を呼ぶ

さらに、`src/shaders/heatmapVertex.ts`・`heatmapFragment.ts`内のシェーダーTODOも実装してください(`uSensorBoost`を使った点のサイズ変化・色の変化)。

```

## よくあるエラー

- `aIntensity`属性の`itemSize`を`1`にし忘れ、色の計算がおかしくなる
- Bloomの`threshold`を調整せずに、ヒートマップの明るい色(赤系)以外まで発光してしまい、見た目がうるさくなる
- `sensorStore`の購読を`main.ts`ではなく`entities`側の中で直接行ってしまい、GUIの`updateMode`(`paused`)が効かなくなる → 購読とモード判定は`main.ts`側で行う
- リサイズ時に`composer`側のサイズ更新を忘れ、ウィンドウサイズ変更後に表示が歪む

attribute float aIntensity;
uniform float uSensorBoost;

varying float vIntensity;

void main() {
  vIntensity = aIntensity;

  vec4 mvPosition = modelViewMatrix * vec4(position, 1.0);

  float baseSize = 40.0 * (1.0 / -mvPosition.z);
  gl_PointSize = baseSize * (0.6 + uSensorBoost * 0.8);

  gl_Position = projectionMatrix * mvPosition;
}
`;

export const heatmapFragmentShader = /* glsl */ `
uniform float uSensorBoost;
varying float vIntensity;

vec3 heatmapColor(float t) {
  vec3 lowColor = vec3(0.15, 0.25, 0.65);
  vec3 midColor = vec3(0.2, 0.7, 0.4);
  vec3 highColor = vec3(1.0, 0.35, 0.2);

  if (t < 0.5) {
    return mix(lowColor, midColor, t / 0.5);
  }
  return mix(midColor, highColor, (t - 0.5) / 0.5);
}

void main() {
  vec2 coord = gl_PointCoord - vec2(0.5);
  if (length(coord) > 0.5) {
    discard;
  }

  float boosted = clamp(vIntensity + uSensorBoost * 0.3, 0.0, 1.0);
  vec3 color = heatmapColor(boosted);

  gl_FragColor = vec4(color, 1.0);
}
`;
```

import * as THREE from 'three';
import type { DepthPoint } from '../utils/parseDepthCsv';
import { heatmapVertexShader } from '../shaders/heatmapVertex';
import { heatmapFragmentShader } from '../shaders/heatmapFragment';

export class HeatmapPointCloud {
  readonly points: THREE.Points<THREE.BufferGeometry, THREE.ShaderMaterial>;
  private readonly uniforms: { uSensorBoost: { value: number } };

  constructor(depthPoints: DepthPoint[]) {
    const geometry = new THREE.BufferGeometry();

    const positions = new Float32Array(depthPoints.length * 3);
    const intensities = new Float32Array(depthPoints.length);

    depthPoints.forEach((point, i) => {
      positions[i * 3] = point.x;
      positions[i * 3 + 1] = point.y;
      positions[i * 3 + 2] = point.z;
      intensities[i] = point.intensity;
    });

    geometry.setAttribute('position', new THREE.Float32BufferAttribute(positions, 3));
    geometry.setAttribute('aIntensity', new THREE.Float32BufferAttribute(intensities, 1));
    geometry.computeBoundingSphere();

    this.uniforms = { uSensorBoost: { value: 0 } };

    const material = new THREE.ShaderMaterial({
      uniforms: this.uniforms,
      vertexShader: heatmapVertexShader,
      fragmentShader: heatmapFragmentShader,
    });

    this.points = new THREE.Points(geometry, material);
  }

  setSensorBoost(value: number): void {
    this.uniforms.uSensorBoost.value = value;
  }
}
```

import * as THREE from 'three';
import { OrbitControls } from 'three/examples/jsm/controls/OrbitControls.js';

export interface SceneContext {
  scene: THREE.Scene;
  camera: THREE.PerspectiveCamera;
  renderer: THREE.WebGLRenderer;
  controls: OrbitControls;
}

export function createScene(container: HTMLElement): SceneContext {
  const scene = new THREE.Scene();
  scene.background = new THREE.Color(0x0a0a12);

  const camera = new THREE.PerspectiveCamera(
    60,
    window.innerWidth / window.innerHeight,
    0.1,
    100
  );
  camera.position.set(0, 3, 8);

  const renderer = new THREE.WebGLRenderer({ antialias: true });
  renderer.setSize(window.innerWidth, window.innerHeight);
  renderer.setPixelRatio(window.devicePixelRatio);
  container.appendChild(renderer.domElement);

  const controls = new OrbitControls(camera, renderer.domElement);
  controls.enableDamping = true;

  return { scene, camera, renderer, controls };
}

import * as THREE from 'three';
import { EffectComposer } from 'three/examples/jsm/postprocessing/EffectComposer.js';
import { RenderPass } from 'three/examples/jsm/postprocessing/RenderPass.js';
import { UnrealBloomPass } from 'three/examples/jsm/postprocessing/UnrealBloomPass.js';

export interface BloomComposer {
  composer: EffectComposer;
  bloomPass: UnrealBloomPass;
  setSize: (width: number, height: number) => void;
}

export function createBloomComposer(
  renderer: THREE.WebGLRenderer,
  scene: THREE.Scene,
  camera: THREE.PerspectiveCamera
): BloomComposer {
  const composer = new EffectComposer(renderer);
  composer.addPass(new RenderPass(scene, camera));

  const bloomPass = new UnrealBloomPass(
    new THREE.Vector2(window.innerWidth, window.innerHeight),
    1.1,
    0.4,
    0.35
  );
  composer.addPass(bloomPass);

  const setSize = (width: number, height: number): void => {
    composer.setSize(width, height);
    bloomPass.setSize(width, height);
  };

  return { composer, bloomPass, setSize };
}

import GUI from 'lil-gui';
import '../style.css';
import rawCsv from '../data/depth-scan.csv?raw';
import { createScene } from './scene/createScene';
import { parseDepthPoints } from './utils/parseDepthCsv';
import { HeatmapPointCloud } from './entities/HeatmapPointCloud';
import { createBloomComposer } from './postprocessing/createBloomComposer';
import { connectSensorSocket } from './utils/connectSensorSocket';
import { sensorStore } from './state/sensorStore';

const appElement = document.getElementById('app');
if (!appElement) {
  throw new Error('#app element not found');
}

const { scene, camera, renderer, controls } = createScene(appElement);

const depthPoints = parseDepthPoints(rawCsv);
const pointCloud = new HeatmapPointCloud(depthPoints);
scene.add(pointCloud.points);

const { composer, bloomPass, setSize } = createBloomComposer(renderer, scene, camera);

connectSensorSocket();

const guiState = { updateMode: 'realtime' as 'realtime' | 'paused' };
const gui = new GUI();

const bloomFolder = gui.addFolder('Bloom');
bloomFolder.add(bloomPass, 'strength', 0, 3, 0.05);
bloomFolder.add(bloomPass, 'radius', 0, 1, 0.01);
bloomFolder.add(bloomPass, 'threshold', 0, 1, 0.01);

gui.add(guiState, 'updateMode', ['realtime', 'paused']).name('sensor updates');

sensorStore.subscribe((state) => {
  if (guiState.updateMode === 'paused') return;
  pointCloud.setSensorBoost(state.sensorValue);
});

window.addEventListener('resize', () => {
  camera.aspect = window.innerWidth / window.innerHeight;
  camera.updateProjectionMatrix();
  renderer.setSize(window.innerWidth, window.innerHeight);
  setSize(window.innerWidth, window.innerHeight);
});

function animate(): void {
  requestAnimationFrame(animate);
  controls.update();
  composer.render();
}

animate();
```
