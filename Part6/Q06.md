# Part6 | センサー値がリアルタイムに動くと点群の色が変わるデモを作ろう

## 目標

WebSocketで受信したリアルタイムデータを、状態管理層を経由してThree.jsシーンに反映する設計ができるようになる。

## 今回の演習で学ぶこと

### WebSocketによるリアルタイム通信

HTTPは「リクエストを送って、レスポンスを受け取ったら終わり」という通信ですが、WebSocketは接続を維持したまま双方向にデータをやり取りできます。センサーの値のように「サーバー側から継続的にプッシュされてくるデータ」を扱うのに適しています。

今回は`server/mock-sensor-server.mjs`という、疑似的なセンサー値(0〜1の間でサイン波のように変動する値)を0.5秒間隔で送り続けるモックサーバーをあらかじめ用意しています。`npm run dev`を実行すると、Viteの開発サーバーとこのモックサーバーが`concurrently`によって同時に起動します。

### 受信データを直接オブジェクトに書き込まない設計

WebSocketの`onmessage`の中で直接`mesh.material.color.set(...)`のようにThree.jsのオブジェクトを書き換えることもできますが、これだとWebSocketの受信処理がThree.jsの内部構造を直接知っている状態になり、見通しが悪くなります。

そこでPart1と同じように、「WebSocket受信 → ストアの状態を更新 → 3Dオブジェクト側がストアを購読して反映」という一方向の流れを作ります。この設計にしておくと、後から「WebSocketではなくGUIで手動で値を変える」ようにしたくなった場合も、ストアを書き換えるだけで済み、3Dオブジェクト側のコードは変更不要になります。

### 受信頻度とレンダリング頻度の違い

センサー値は0.5秒に1回しか届きませんが、`renderer.render()`は60fps(約16msに1回)で呼ばれ続けます。今回は「値が届いたときだけ色を更新する」というシンプルな実装にしていますが、受信頻度がレンダリング頻度よりずっと低い場合、値の変化がカクついて見えることがあります(発展的な課題として、値の補間について考えてみるのも良い経験になります)。

## 問題文

`src/main.ts`には、球状に配置した点群の表示・OrbitControls・GUIの土台が実装済みです。以下の3つの未実装(TODO)を実装し、「センサー値がリアルタイムに動くと点群の色が変わる」デモを完成させてください。

1. `SensorStore`クラスの`setState`・`subscribe`を実装し、シンプルなPub/Subストアとして機能するようにする
2. `intensityToColor`を実装し、0〜1のセンサー値を色(青→緑→赤のグラデーション)に変換できるようにする
3. `connectSensorSocket`を実装し、`ws://localhost:8081`のモックサーバーに接続して、受信した値を`sensorStore.setState(...)`に反映する(接続が切れた場合は再接続を試みる処理も入れる)
4. ストアを購読し、GUIの`updateMode`が`realtime`のときだけ、受信値の色を点群に反映する

### 完成イメージ

```
- 球状に並んだ点群が画面に表示される

- GUIパネルで updateMode を "paused" にすると、色の更新が止まる
```

## ヒント

- `SensorStore`の`setState`は、Part1の`BoxStore`と同じパターンで実装できます(`this.state = {...this.state, ...partial}`のあと、登録されている購読者全員に新しい状態を渡して呼び出す)
- `subscribe`は、購読者(コールバック関数)を`Set`などに登録し、「登録解除するための関数」を返すようにします
- WebSocketの`message`イベントでは`event.data`が文字列なので、`JSON.parse`でオブジェクトに変換します。受信データが期待した形かどうかは、型ガード関数でチェックすると安全です
- 再接続処理は、`socket.addEventListener('close', () => { setTimeout(connectSensorSocket, 2000); })`のように、closeイベントで再度接続関数を呼び出す形で実装できます

## 達成基準

- [ ] `SensorStore`がPub/Subとして正しく機能している
- [ ] WebSocketからの受信データが、直接3Dオブジェクトを書き換えず、ストア経由で反映されている
- [ ] GUIの`updateMode`切り替えが機能する
- [ ] WebSocketが切断された場合、自動的に再接続を試みる

## よくあるエラー

- モックサーバーが起動する前にクライアントが接続を試みて失敗する → `close`/`error`イベントでの再接続処理が入っていれば、数秒後に自動的につながる
- `JSON.parse`した結果を型チェックせずに使い、想定外のデータ形式が来たときにエラーになる → 型ガード関数で確認してから使う
- ストアの`subscribe`で登録した購読者を解除する関数を実装し忘れ、コンポーネントが増えたときにメモリリークの原因になる(今回の規模では実害はないが、実務では重要な観点)
- `updateMode`が`paused`のときも`setSensorValue`を呼んでしまい、GUIの効果が出ない → ストアの購読コールバック内でモードを判定する

## チャレンジ課題(任意)

受信値をそのまま反映するのではなく、前回値と新しい値の間を毎フレーム少しずつ近づける(線形補間)ことで、色の変化を滑らかにしてみよう。

```ts
export interface SensorState {
  sensorValue: number;
}

type Listener = (state: SensorState) => void;

class SensorStore {
  private state: SensorState = { sensorValue: 0.5 };
  private listeners = new Set<Listener>();

  getState(): SensorState {
    return this.state;
  }

  setState(partial: Partial<SensorState>): void {
    this.state = { ...this.state, ...partial };
    this.listeners.forEach((listener) => listener(this.state));
  }

  subscribe(listener: Listener): () => void {
    this.listeners.add(listener);
    return () => this.listeners.delete(listener);
  }
}

export const sensorStore = new SensorStore();
```

```ts
import * as THREE from 'three';

const LOW_COLOR = new THREE.Color(0x2b6cb0);
const MID_COLOR = new THREE.Color(0x38a169);
const HIGH_COLOR = new THREE.Color(0xe53e3e);

export function intensityToColor(value: number): THREE.Color {
  const clamped = THREE.MathUtils.clamp(value, 0, 1);
  const color = new THREE.Color();

  if (clamped < 0.5) {
    color.lerpColors(LOW_COLOR, MID_COLOR, clamped / 0.5);
  } else {
    color.lerpColors(MID_COLOR, HIGH_COLOR, (clamped - 0.5) / 0.5);
  }

  return color;
}
```

```ts
import { sensorStore } from '../state/sensorStore';

const SOCKET_URL = 'ws://localhost:8081';
const RECONNECT_DELAY_MS = 2000;

interface SensorPayload {
  timestamp: number;
  sensorValue: number;
}

function isSensorPayload(value: unknown): value is SensorPayload {
  return (
    typeof value === 'object' &&
    value !== null &&
    typeof (value as SensorPayload).timestamp === 'number' &&
    typeof (value as SensorPayload).sensorValue === 'number'
  );
}

export function connectSensorSocket(): void {
  const socket = new WebSocket(SOCKET_URL);

  socket.addEventListener('message', (event: MessageEvent<string>) => {
    const parsed: unknown = JSON.parse(event.data);
    if (isSensorPayload(parsed)) {
      sensorStore.setState({ sensorValue: parsed.sensorValue });
    }
  });

  socket.addEventListener('close', () => {
    console.warn(`sensor socket closed, retrying in ${RECONNECT_DELAY_MS}ms...`);
    setTimeout(connectSensorSocket, RECONNECT_DELAY_MS);
  });

  socket.addEventListener('error', () => {
    socket.close();
  });
}
```

```ts
import * as THREE from 'three';
import { intensityToColor } from '../utils/colormap';

export class SensorPointCloud {
  readonly points: THREE.Points<THREE.BufferGeometry, THREE.PointsMaterial>;

  constructor(pointCount: number) {
    const geometry = new THREE.BufferGeometry();
    const positions = new Float32Array(pointCount * 3);

    for (let i = 0; i < pointCount; i++) {
      const phi = Math.acos(1 - (2 * (i + 0.5)) / pointCount);
      const theta = Math.PI * (1 + Math.sqrt(5)) * i;
      const radius = 2.5;

      positions[i * 3] = radius * Math.sin(phi) * Math.cos(theta);
      positions[i * 3 + 1] = radius * Math.sin(phi) * Math.sin(theta);
      positions[i * 3 + 2] = radius * Math.cos(phi);
    }

    geometry.setAttribute('position', new THREE.Float32BufferAttribute(positions, 3));
    geometry.computeBoundingSphere();

    const material = new THREE.PointsMaterial({ size: 0.08, color: 0x4f8fdb });
    this.points = new THREE.Points(geometry, material);
  }

  setSensorValue(value: number): void {
    this.points.material.color.copy(intensityToColor(value));
  }
}
```

```ts
import * as THREE from 'three';
import { OrbitControls } from 'three/examples/jsm/controls/OrbitControls.js';

export interface SceneContext {
  scene: THREE.Scene;
  camera: THREE.PerspectiveCamera;
  renderer: THREE.WebGLRenderer;
  controls: OrbitControls;
}

export function createScene(container: HTMLElement): SceneContext {
  const scene = new THREE.Scene();
  scene.background = new THREE.Color(0x0a0a12);

  const camera = new THREE.PerspectiveCamera(
    55,
    window.innerWidth / window.innerHeight,
    0.1,
    100
  );
  camera.position.set(0, 0, 6);

  const renderer = new THREE.WebGLRenderer({ antialias: true });
  renderer.setSize(window.innerWidth, window.innerHeight);
  renderer.setPixelRatio(window.devicePixelRatio);
  container.appendChild(renderer.domElement);

  const controls = new OrbitControls(camera, renderer.domElement);
  controls.enableDamping = true;

  window.addEventListener('resize', () => {
    camera.aspect = window.innerWidth / window.innerHeight;
    camera.updateProjectionMatrix();
    renderer.setSize(window.innerWidth, window.innerHeight);
  });

  return { scene, camera, renderer, controls };
}
```
